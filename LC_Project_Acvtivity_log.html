Important Piece of Code

This code is crucial to my second what if question. I had previously figured out my percentage issue which meant that creating this code was not as difficult as I had anticipated. With this code working it meant that I could display these values to the user and also send them to my graphing code so it could be shown to the user visually. As you can see all the times that study took place, all the wasted minutes in each session and the overall study time per session. This function find the pair of waste and total minutes that give the lowest percentage and it is then matched with the time these values came from. This works as when all these items were added to their respective lists they were done chronologically. This meant that I could use indexing to match these values
