This is what went into my planning process

When I began my brainstorming process, I was initially drawn to something that could link well-being and studying. I know from personal experience and stories from peers that study can be a major stress inducer for students. I want to take advantage of the micrbobit's versatility. I want to be able to utilize its ability to pick up sound levels and find a way to manipulate its features into making a clock. The buttons on the micrbobit will also be of great use to me as I can make the user press these buttons when required. My plan is to take in the study location from the user and from there on the study session is to begin. If their studying environment surpasses the set sound threshold, at the end of each minute a beep will be made alerting the student that that minute was ineffective encouraging them to seek a new place to study. On the other hand, if it was an effective minute no sound will be made so as not to disturb the student. Once their study session has been completed, they will be required to click a button on the microbit so that the data from that session can be logged. From there, it will be sent to the firebase, taken in by another code file to be interpreted and displayed to fulfill my basic and advanced requirements. I also plan to have comments throughout the code so that its easy to follow in the case that i have to take time away from the project. To record my video I will be using a combination of OBS and my phones camera. I will use the microphone from my headphones to record the voice over. It will all be editted in OpenShot.

Here is what went into my designing process

This project will consist of 6 main parts, User input, a data logging system, a data analysis system, a prediction model, interpretation functions and graphing functions:

  1. The user will use the microbit and will have the data logged to firebase so that my model can use the info from each session to interpret it
  2. A dataset will be added to the firebase so that there is data that can be interpreted in order to give the user an idea of how it is all laid out and their own personal study session data will be interpreted in unison with this dataset to show how they compared to the example data
  3. All the data at the given time will be analyzed and interpreted in both words and visually as to give the user the best chance of understanding the data
  4. Depending on the information the user wishes to see a prediction will be made in text within the data they are presented with

I will meet some of the universal design features by trying to encorporate principals 1 and 7
